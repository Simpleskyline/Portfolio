About Me

I'm a full-stack developer and designer based in Kenya, specializing in creating modern web applications that help businesses thrive. From pixel-perfect frontends to robust backends, I work across the full spectrum of modern web technologies.

My expertise spans across frontend frameworks like React and Next.js, backend technologies including Django and Node.js, and modern tools like Figma, Vercel, and Supabase. I focus on delivering fast, scalable, production-ready solutions tailored to local businesses.

Whether you need a simple landing page or a complex e-commerce platform, I'm here to help transform your vision into reality with affordable, high-quality web solutions.
